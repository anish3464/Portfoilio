🚘 Problem Solved

Fabro Leather needed a comprehensive solution to:

  • Track complaints from car manufacturers related to seats
  • Manage dimensional data with CAD files
  • Control access based on user roles (admin, engineer, manager)
  • Allow secure media uploads from the field

✨ Key Features

📋 Complaint Management

  • Add/view/edit/delete complaints
  • Filter/search by channel, date, or car details
  • Inline media preview (image & video)
  • Status workflows (open → in progress → resolved)

🚗 Car Hierarchy Database

  • Brand → Model → Year → Sub-Model
  • Unique layout codes with seat/door configs
  • Auto-detection of duplicates with modal warnings

📱 Media Upload & Preview

  • Multiple images/videos per complaint
  • Inline previews, hover video playback
  • Files stored on AWS S3 per complaint folder

🔒 Security & Access Control

  • Role-based Access Control (RBAC)
  • Signed S3 URLs for file access
  • Audit logging of user actions

💡 Challenges & Solutions

Dynamic Dropdowns: Managing dependent dropdowns across multiple models
S3 Integration: Efficient per-complaint folder organization
Duplicate Prevention: Fuzzy matching to avoid duplicate car entries
Video Optimization: Hover playback without heavy bandwidth usage

Carcinogen Detector – Product Safety Analyzer

AI-driven tool to identify carcinogenic ingredients using OCR + deep learning

🔧 Tech Stack

AI & ML

ChemBERTa LLaMA 3.2 HuggingFace

Backend

Python Flask Tesseract OCR

Data Sources

PubChem API SMILES IARC Data

🧪 Problem Solved

Many cosmetics, shampoos, and food items contain chemicals flagged as potential carcinogens. Most consumers are unaware of these risks.

This project allows users to scan a product label and get a detailed carcinogen risk report powered by LLMs and chemical datasets.

✨ Key Features

📷 OCR Ingredient Extraction

  • Extract ingredients from product label images
  • Clean and standardize chemical names
  • Handle multiple languages and formats

🧬 SMILES Mapping

  • Fetch chemical structures from PubChem
  • Generate molecular fingerprints
  • Resolve ambiguous chemical names

🤖 AI Classification

  • Fine-tuned ChemBERTa for carcinogen prediction
  • Training on IARC/NIH labeled datasets
  • Confidence scoring and risk levels

💬 LLM Explanations

  • LLaMA 3.2 explains why chemicals are flagged
  • Plain-language risk explanations
  • Source citations and confidence scores

💡 Technical Challenges

Name Standardization: Inconsistent ingredient names across regions/products
PubChem Resolution: Multiple matches for similar chemical names
Limited Training Data: Tuning ChemBERTa on sparse labeled datasets
LLM Grounding: Preventing hallucinations in explanations
